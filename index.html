⚠️ 重要提示：安装 Allow CORS 插件

由于浏览器的同源策略限制，直接访问外部 DNS 服务器会被阻止。请安装 Allow CORS 插件以正常使用测速功能。

点击图标安装：
Chrome Firefox Edge
1 点击上方图标安装对应浏览器的插件
2 安装后插件图标显示为灰色 C，表示关闭状态
3 点击插件图标，图标变为橙色 C，即开启 CORS
4 测试完成后建议再次点击图标关闭插件

Allow CORS 插件状态指示

📌 该插件无设置选项，仅有一个开关按钮。图标灰色=关闭，橙色=开启。
